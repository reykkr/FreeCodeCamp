HTML

HTML stands for Hypertext Markup Language..

It is the standard markup language for creating and structuring web pages.

HTML uses tags to define the structure and content of a web page.

HTML5 is the latest version of HTML, introducing new elements and features.

It is essential for building the foundation of websites and web applications.

<h1>Hello, World!</h1> <p>This is a paragraph.</p>
  • Tags: HTML uses tags to define elements and structure content, such as p for paragraphs and img for images.
  • Attributes: HTML elements can have attributes like "src" for specifying image sources or "href" for links.
  • Headings: HTML offers six levels of headings (h1 to h6) to define the hierarchical structure of the content.
  • Lists: HTML supports ordered (ol) and unordered (ul) lists for organizing items.
  • Links: a tags are used to create hyperlinks that connect web pages and resources.
CSS

CSS stands for "Cascading Style Sheets" and is a stylesheet language used to control the presentation and layout of HTML documents.

CSS allows web developers to apply styles such as colors, fonts, margins, and positioning to HTML elements.

It follows a cascade hierarchy, where styles can be defined in multiple places, and the most specific rule takes precedence.

CSS3 is the latest version of CSS and introduced many new features like transitions, animations, and media queries for responsive design.

CSS can be included in an HTML document using internal styles, inline styles, or by linking an external CSS file.

body { background-color: #f0f0f0; } h1 { color: #333; }
  • Selectors: Used to target HTML elements for styling, such as class selectors (e.g., ".classname") or element selectors (e.g., "p").
  • Properties: The styling attributes applied to the selected elements, like "color," "font-size," "margin," etc.
  • Values: The specific settings assigned to the properties, such as "red" for color or "20px" for margin.
  • Box Model: Defines how elements are rendered, consisting of content, padding, border, and margin.
  • Media Queries: Used to apply different styles based on the device screen size or other characteristics, enabling responsive design.
JavaScript and Java

JavaScript

JavaScript is a high-level, interpreted scripting language primarily used for web development.

It was created by Brendan Eich in 1995 and originally named "LiveScript" before being renamed to "JavaScript."

JavaScript is often used to add interactivity and dynamic content to websites, enabling client-side functionality.

Node.js allows JavaScript to be used on the server-side, facilitating full-stack development.

Despite its name, JavaScript is unrelated to Java and has different syntax and use cases.

console.log("Hello, World!");
  • Scripting language
  • Used for web development
  • Client-side interactivity
  • Node.js for server-side
  • Different from Java

Java

Java is a high-level, object-oriented programming language developed by James Gosling and his team at Sun Microsystems in the early 1990s.

It was designed to be platform-independent, allowing "write once, run anywhere" (WORA) capabilities through the Java Virtual Machine (JVM).

Java is widely used for enterprise-level applications, Android app development, and various other software projects.

Java's syntax is similar to C++, making it familiar to developers with a background in C-based languages.

Java applications are compiled into bytecode, which is executed by the JVM at runtime.

System.out.println("Hello, World!");
  • Object-oriented language
  • Platform-independent
  • JVM enables WORA
  • Popular for enterprise and Android
  • Compiled into bytecode
Python

Python is a high-level, interpreted programming language.

  • It was created by Guido van Rossum and first released in 1991.

    Python emphasizes code readability and simplicity.

    It supports multiple programming paradigms, including procedural, object-oriented, and functional programming.

    Python is a high-level, interpreted programming language

    Python has a large standard library and a thriving community with extensive third-party packages.

    greeting = 'Hello, World!' print(greeting)
    • Simplicity: Python emphasizes readability and straightforward syntax, making it easy to learn and understand.
    • Dynamically Typed: Python is dynamically typed, meaning variable types are determined at runtime, allowing for flexible and concise code.
    • Interpreted: Python code is executed line-by-line by an interpreter, enabling rapid development and easier debugging.
    • High-Level Data Structures: Python provides built-in support for lists, tuples, dictionaries, and sets, making complex data manipulation simpler.
    • Extensive Standard Library: Python comes with a vast collection of modules and libraries, providing a wide range of functionalities for various tasks.
